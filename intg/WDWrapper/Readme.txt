Variables
----------
RECMainClass.java in src/bnw.abm.intg.sync.wdwrapper
	Executes WeddingGrid externally.
	To collect survey data: uncomment either state.getIndividualDataShortRelations() or state.getIndividualDataLongRelations()
	Change output file name string to survey/RelationshipStatus_short_ or survey/RelationshipStatus_long_ accordingly. Otherwise population megrge will not work

Adjusting size of the population
	Population size has to be change ins paramters.xml in WeddingGrid project's WeddingGrid.rs directory.
